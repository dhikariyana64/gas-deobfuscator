\# GAS Deobfuscator



A JavaScript deobfuscator for Google Apps Script projects obfuscated with javascript-obfuscator.



\## Features



\- Parse JavaScript

\- Detect javascript-obfuscator

\- Recover string arrays

\- Inline decoded strings

\- Remove decoder functions

\- Pretty-print output



\## Installation



```bash

npm install

```



\## Usage



```bash

node src/cli.js fixtures/Code-gs.txt

```
